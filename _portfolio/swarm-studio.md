Swarm Studio: Reservoirs is an interactive browser-based experiment for exploring swarm-based reservoir computing through sound and motion.

It lets you drive the system with different input signals, observe how the collective dynamics respond, and inspect traces such as grouping behavior in real time. I built it as a public-facing sandbox for the research behind "State transitions unlock temporal memory in swarm-based reservoir computing," by Tanner Lund, Alyssa Adams, Nathanael Aubert-Kato, and Takashi Ikegami.
